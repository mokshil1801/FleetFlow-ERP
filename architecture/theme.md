# SOP: Visual System & Theme

## Color Tokens
- **Success/Available:** `bg-green-100 text-green-800` (Hex: #10B981)
- **Active/On Trip:** `bg-blue-100 text-blue-800` (Hex: #3B82F6)
- **Warning/Pending:** `bg-orange-100 text-orange-800` (Hex: #F59E0B)
- **Error/In Shop:** `bg-red-100 text-red-800` (Hex: #EF4444)
- **Neutral/Retired:** `bg-gray-100 text-gray-800` (Hex: #6B7280)

## Layout Constraints
- **Sidebar:** Fixed width (256px), collapsible icons.
- **Cards:** White background, rounded corners (rounded-lg), subtle shadow (shadow-sm).
- **Typography:** Inter or Outfit (Sans-serif). Titles in semi-bold.

## Animations
- **Transitions:** `ease-in-out` for modal fade-ins.
- **Micro-interactions:** Scale effect on hover for KPI cards.
